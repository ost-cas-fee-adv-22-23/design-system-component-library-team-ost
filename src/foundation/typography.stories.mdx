# Heading Typography

**Sizes:** 2xl(1.5rem), 3xl(2rem), 4xl(2.5rem), 5xl(3rem)

**Weights:** 600(semibold), 700(bold)

## Usage
The heading component can be used as H1-H4 headings. [Documentation](/story/components-typography--headings).

The text color is inherited from the parent element and has to be set outside the component.

# Label Typography

**Sizes:** xs(0.875rem), sm(1rem), lg(1.25rem), 2xl(1.5rem)

**Weights:** 500(medium), 600(semibold)

<Typeset
  fontSizes={[
