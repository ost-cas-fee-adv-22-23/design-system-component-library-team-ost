# Typography
Mumble uses the google font "Poppins" as primary font. Poppins is a contemporary sans-serif that's accessible and with a smooth typeface. It's ideal for short texts like mumbles and for headlines. Poppins Medium is used for text and Poppins Semibold and Bold for headings and interactions.

# Headings

**Font:** Poppins

**Sizes:** 2xl(1.5rem), 3xl(2rem), 4xl(2.5rem), 5xl(3rem)

**Weights:** 600(semibold), 700(bold)

## Usage
The heading component can be used as HTML headings. It creates depending on the size a "h1"-"h4" HTML Element. [Component documentation](/story/components-typography--default-heading).

### Color
The color of the headline refers to the current text color of the parent element. So you could easily set the desired color of the icon outside the component. In the following example the color is set on the parent div element of the icon.
```jsx
<div className="text-slate-700">
  <Headline />
</div>
```

  fontFamily={typography.type.primary}
/>

## Usage
The label component has two uses. It can be used as a label for form items (input, textarea, ...), or also for labeling buttons, links, help texts, etc. Depending on the use case it creates a "span" or "label" HTML element. [Component documentation](/story/components-typography--default-label).

### Color
The color of the label refers to the current text color of the parent element. So you could easily set the desired color of the icon outside the component. In the following example the color is set on the parent div element of the icon.
```jsx
<div className="text-slate-700">
  <Label />
</div>
```

### Type
If an htmlFor parameter is supplied, the component creates a label HTML element. Without hmtlFor parameter, a "span" HTML element is created.
```jsx
  <Label htmlFor={id} />
</div>
```
