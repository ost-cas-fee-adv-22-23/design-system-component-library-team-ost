# Breakpoints

We use the default [tailwind breakpoints](https://tailwindcss.com/docs/responsive-design).

| **Name**       | **Breakpoint [Pixels]**  | **Usage**                                  |
| :------------- | :----------------------- | :----------------------------------------- |
| **mobile**     | 0                        | Smartphones                                |
| **sm**         | 640                      | Large smartphones, small tablets           |
| **md**         | 768                      | Tablets portrait                           |
| **lg**         | 1024                     | Tablets landscape                          |
| **xl**         | 1280                     | Small screens                              |
| **2xl**        | 1536                     | Larger screens                             |


## Storybook viewports
To test the components in different viewports we customized the storybook viewports for our needs.
