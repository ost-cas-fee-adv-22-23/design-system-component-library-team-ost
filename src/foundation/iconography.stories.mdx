## Usage
Each icon is a single react component. All you have to do is to import the desired icon and start use it in your code. You can try a single icon component [here](/story/components-icon--icon).

The color of the icon refers to the current text color of the element. So you could easily set the desired color of the icon outside the component. In the following example the color is set on the parent div element of the icon.
