## Usage
Each logo is a single react component. All you have to do is to import the desired icon and start use it in your code. You can try a single icon component [Component](/story/components-logo--logo-with-click).
```jsx 
import { MumbleViolet } from './mumble-violet';
```

### Click action
By passing a onClick function, the icon will be clickable.
```jsx 
<MumbleViolet onClick={()=> {}} />
```
