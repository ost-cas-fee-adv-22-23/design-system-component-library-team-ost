# Colors

The Color Palette is based on the Tailwind colors "Slate", "Pink" and "Violet".

## Color Slate
Our primary text color "Slate", can be used for links, input, placeholder. It's designed to be accessible for text with slate-600 and above.
Slate-100 is also used as page background.

The gradient from pink to violet should be used sparingly. For example, as a mood background or for special buttons that should stand out from primary buttons.
